# Google TP
### Description
Ce projet est une reproduction fidèle de la page d'accueil de Google. Réalisé dans le cadre d'un Travaux Pratiques (TP), l'objectif principal était de maîtriser les concepts fondamentaux du positionnement CSS (Flexbox), la structuration sémantique HTML et l'esthétique du design minimaliste.

### Fonctionnalités
Barre de navigation (Header) : Liens Gmail et Images avec effets au survol.

Section centrale (Main) : Logo Google officiel et barre de recherche interactive.

Barre de recherche : Design arrondi avec icônes de recherche et micro, incluant un effet d'ombre au survol (box-shadow).

Boutons d'action : Boutons "Recherche Google" et "J'ai de la chance" stylisés selon la charte graphique.

Design Responsive : Mise en page centrée verticalement et horizontalement.

### Technologies Utilisées
HTML5 : Structuration sémantique du contenu.

CSS3 :

Flexbox pour l'alignement précis.

Réinitialisation (Reset) des styles par défaut.

Gestion des pseudo-classes (:hover) pour l'interactivité.

Google Fonts / Arial pour la typographie.

### Apprentissages
Au cours de ce TP, j'ai pu approfondir :

L'utilisation de display: flex pour centrer des éléments de manière complexe.

La gestion des images et des icônes distantes.

La création d'une barre de recherche moderne sans utiliser de frameworks externes (pur CSS).
